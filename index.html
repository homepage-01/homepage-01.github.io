⒈　ホームページってどうやって作るの？

  • ホームページの基本は"HTML"

    　HTMLはHyperText Markup Languageの略語で、ホームページの基礎となるものです。

    これを使うことによって普段私たちが目にするような見出しやリンクのついたホームページを作ることができます。

    GoogleやYahoo、さらに今あなたが見ているこのサイトももちろんHTMLが使われています。

  • ホームページを彩るのは"CSS"

    一通りの情報は書きつくしたがどこか見づらく物足りない

    そんな問題を解決するのがCSS（Cascading Style Sheetの略称）です

    CSSによってWebページの文字の色や大きさ、背景、配置といったスタイル（見た目）を設定し、ホームページを見やすく、より魅力的に変えることができます

    CSSを使うことによって印象がガラッと変わりますね！

    今回は時間の都合上CSSの説明をすることはできませんが、代わりにCSSを基礎から学べるYoutubeの動画を貼り付けておきます。

Lesson :HTMLの仕組みを知ろう！

まずはHTMLから学習していきましょう！

HTMLでは、テキストに「タグ」と呼ばれる目印をつけています。これによりテキストに「見出し」や「リンク」などの意味づけがされます。

タグには「開始タグ＜〇＞」と「終了タグ＜/〇＞」があり、終了タグには"/"が必要です。忘れないように気を付けましょう。

最初はこれらのタグを使う練習をしてみましょう！

今回はホームページ作成によく使われる４つの要素について説明します！

  1. 見出しと段落

    2. 見出しには＜h〇＞タグを使います。＜h1＞から＜h6＞まであり、数字が大きくなるにつれて見出しが小さくなっていきます。ｐは段落を作成するためのタグで見出し以外の文章は基本的にこれを使います。

    この２つの要素を組み合わせて、大切なところを強調したり、見出し同士のつながりを作ったりしてみましょう

  2. リンク

    3. ホームページ上にリンクを作成するためには＜a＞タグを使います。

    開始タグと終了タグの間に記述をしないと何も表示されなくなるので気をつけましょう。

    せっかくなのでここでホームページ上へのYoutubeの動画の埋め込み方を学びましょう

    好きなYouTubeの動画をクリック➡共有ボタンをクリック➡埋め込むをクリックしてコピーする

    コピーしたものをそのまま貼り付ければ動画がそのままホームページ上に埋め込むことができます

  3. 画像

    4. ホームページ上に画像を表示するためには＜img＞タグを使います。ダブルクォーテーション内に画像のリンクを指定することで画像が表示されます。

    また、＜img＞要素はテキストを囲むことはないので終了タグは必要ありません。

    Web上だけでなく自分のフォルダーからも画像を持ってくることができます。

  4. リスト

    5. ホームページ上でリストを作るには＜li＞、＜ul＞、＜ol＞などのタグを使います

    ＜li＞要素は、囲む要素によって種類が変わるという特徴があります。例えば＜ul＞で囲むと黒点が先頭につき、＜ol＞で囲むと数字が連番でつきます。
